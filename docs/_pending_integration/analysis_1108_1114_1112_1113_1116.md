# Analysis of Recent Refactoring Tasks

## Summary
This document summarizes the changes made during the execution of Tasks 1108, 1112, 1113, 1114, and 1116. The primary goal was to reduce code bloat, improve modularity, and consolidate related functionality in both frontend and backend systems.

## Task 1108: Refactor `css/components/viewer.css`
**Objective:** De-bloat `viewer.css` by extracting logical groups.
**Changes:**
- Extracted hotspot-related styles into `css/components/viewer-hotspots.css`.
- Extracted UI-related styles into `css/components/viewer-ui.css`.
- Updated `viewer.css` to import these new files.
**Result:** Improved maintainability of viewer styles by separating distinct concerns.

## Task 1114: Surgical Refactor SYSTEMS FRONTEND
**Objective:** De-bloat complex frontend systems by extracting logic into `*Logic.res` modules.
**Changes:**
- **`src/systems/Api.res`**: Extracted API definition and client logic into `src/systems/ApiLogic.res`.
- **`src/systems/ExifReportGenerator.res`**: Extracted report generation logic (Extraction, Location, Groups) into `src/systems/ExifReportGeneratorLogic.res`.
- **`src/systems/HotspotLine.res`**: Extracted rendering and math logic into `src/systems/HotspotLineLogic.res`.
- **`src/systems/Simulation.res`**: Extracted navigation and path generation logic into `src/systems/SimulationLogic.res`.
- **`src/systems/Teaser.res`**: Extracted recording, playback, and server communication logic into `src/systems/TeaserLogic.res`.
**Result:** Significant reduction in LOC for the main system files, making them cleaner orchestrators.

## Task 1112: Surgical Refactor SRC BACKEND
**Objective:** De-bloat `backend/src/pathfinder.rs`.
**Changes:**
- Created `backend/src/pathfinder/` directory.
- Extracted graph data structures into `backend/src/pathfinder/graph.rs`.
- Extracted pathfinding algorithms (Walk, Timeline) into `backend/src/pathfinder/algorithms.rs`.
- Updated `backend/src/pathfinder.rs` to re-export these modules.
**Result:** Better organization of the pathfinding domain logic.

## Task 1113: Merge Folders BACKEND
**Objective:** Consolidate fragmented backend folders.
**Changes:**
- Merged `backend/src/services/auth/` (mod + jwt) into a single file `backend/src/services/auth.rs`.
- Merged `backend/src/services/geocoding/` (mod + logic) into a single file `backend/src/services/geocoding.rs`.
- Merged `backend/src/middleware/` (mod + auth + quota + tracker) into a single file `backend/src/middleware.rs`.
- Updated `backend/src/main.rs`, `backend/src/lib.rs`, and `backend/src/services/mod.rs` to reflect these changes.
**Result:** Reduced file system clutter and improved cohesion for small services.

## Task 1116: Classify Map Entries
**Objective:** Update `MAP.md` with new modules.
**Changes:**
- Classified all unmapped modules from the "Unmapped Modules" section into their respective categories (Core Architecture, System Layer, Backend API, etc.).
