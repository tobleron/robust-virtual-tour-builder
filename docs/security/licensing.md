## Dual License Model

Robust Virtual Tour Builder uses a **dual licensing model** to balance open source accessibility with sustainable development:

1. **AGPL v3** - Free for individuals, hobbyists, and open source projects
2. **Commercial License** - Paid, for brokerages and organizations wanting proprietary use

---

## 1. AGPL v3 License (FREE)

**Perfect for:**
- ✅ Individual real estate agents
- ✅ Developers and hobbyists
- ✅ Students and educators
- ✅ Personal projects
- ✅ Open source contributions

### What You Get

- **Unlimited Use:** Create as many virtual tours as you want
- **All Features:** Full access to all functionality
- **Regular Updates:** Benefit from ongoing improvements
- **Community Support:** Access to GitHub issues and discussions

### Requirements

If you use this software under AGPL v3:

- **Share Modifications:** If you modify the code, you must share your changes
- **Network Use:** If you run a modified version as a service, offer source to users
- **Keep License:** Include AGPL license with any distributions
- **State Changes:** Mark modified files with change notices

### What This Means

**Individual Broker Example:**
```
You: Individual real estate agent
Use: Creating tours for your own listings
Pay: FREE
Requirement: If you modify the code, share your changes
```

**Developer Example:**
```
You: Software developer
Use: Learning 360° photography, personal projects
Pay: FREE
Requirement: None (unless you modify and distribute)
```

## Frequently Asked Questions

### General

**Q: I'm an individual broker. Do I need to pay?**  
A: No! Individual brokers can use the software for free under AGPL v3. You only need a commercial license if you're a brokerage with multiple agents or creating tours for clients as a business.

**Q: What's the difference between AGPL and commercial?**  
A: AGPL requires you to share any modifications you make to the code. Commercial license lets you use the software proprietarily without sharing changes.

**Q: Can I switch from AGPL to commercial later?**  
A: Yes! Contact us and we'll apply your usage toward a commercial license.

**Q: I'm a developer building my portfolio. Can I use this free?**  
A: Absolutely! AGPL v3 is perfect for developers. Just share any improvements you make.

### Individual Brokers

**Q: I'm a solo agent creating tours for my listings. Am I "commercial"?**  
A: No, you can use it free under AGPL v3. You're only required to pay if you're a brokerage with multiple agents or creating tours for other agents (as a business).

**Q: Should I donate if I use it free?**  
A: Donations are appreciated but not required. If this software helps your business, consider buying us a coffee ($5-10) or becoming a supporter ($25/month). Every bit helps keep the project free!

☕ [Support the Project](../SPONSORS.md)

**Q: What if I start my own brokerage?**  
A: Once you have multiple agents working under you, you'll need a commercial license. Contact us when you're ready.

### Brokerages

**Q: We're a small brokerage (3 agents). Do we need commercial license?**  
A: Yes, brokerages with multiple agents need a commercial license. Small Brokerage tier is $499/year for up to 10 agents.

**Q: Can we just use AGPL and not share our modifications?**  
A: No, that would violate the AGPL license. If you want to use the software without sharing modifications, you need a commercial license.

**Q: What if one agent in our brokerage wants to use it personally?**  
A: Individual agents can use the free AGPL version for their personal listings. However, if the brokerage is providing it as a tool to agents, you need a commercial license.

### Developers

**Q: Can I use this in my portfolio?**  
A: Yes! AGPL v3 is perfect for portfolio projects. Just credit the project and share any improvements.

**Q: Can I contribute to the project?**  
A: Absolutely! We welcome contributions. See our [Contributing Guide](CONTRIBUTING.md) for details.

**Q: Can I fork this and create my own version?**  
A: Yes, under AGPL v3. Your fork must also be AGPL v3 licensed and you must share modifications.

### Donations & Sponsorship

**Q: How do donations work?**  
A: Donations support ongoing development. We offer:
- One-time donations ($5-10)
- Monthly supporters ($25/month)
- Patrons ($100/month)
- Corporate sponsors (custom)

🏆 [Become a Sponsor](../SPONSORS.md)

**Q: Are donations tax-deductible?**  
A: Currently, no. We're exploring non-profit sponsorship options for the future.

**Q: What do sponsors get?**  
A: Recognition in SPONSORS.md, priority support, early access to features, and input on roadmap (depending on tier).

## Compliance

### How We Enforce

We use a **trust but verify** approach:

1. **Clear License Terms:** Most people comply when terms are clear
2. **Community Reporting:** Users report commercial abuse
3. **Website Monitoring:** We check brokerage websites for unlicensed use
4. **Grace Period:** 30-day cure period for unintentional violations

### What Happens If You Violate

1. **First Notice:** Friendly email explaining license requirements
2. **Second Notice:** Formal cease and desist if no response
3. **Third Notice:** Legal action if still unresolved

### Voluntary Compliance

Realized you've been using it without proper license?

- Contact us within 30 days
- We'll offer grace period
- No penalties for voluntary compliance

### Reporting Violations

Suspect commercial abuse? Report anonymously:
- Email: compliance@robust-vtb.com
- Include: URL, company name, description of use
