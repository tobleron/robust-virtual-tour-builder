# v4.2.18 CSS Styling & UI Behavior Report

This report extracts every visual feature, style, and UI behavior defined in the CSS of Version 4.2.18. This version represents the "perfect" look and feel that the user wants to restore.

## 1. Brand Identity & Design Tokens
*   **Premium Palette:**
    *   `--primary`: #003da5 (Remax Blue)
    *   `--accent`: #ffcc00 (Remax Gold)
    *   `--success`: #10b981 (Emerald 500)
    *   `--danger`: #dc3545 (Remax Red)
    *   `--slate-900`: #0f172a (Deep Navy background)
*   **Shadows:** 
    *   `--shadow-premium`: `0 25px 50px -12px rgba(0, 0, 0, 0.5)` - Creates depth for the viewer and modals.
*   **Glassmorphism:** 
    *   `--glass-bg`: `rgba(15, 23, 42, 0.7)`
    *   `--glass-blur`: `12px`
    *   Used extensively in panels, toasts, and overlays for a modern, layered feel.

## 2. The Viewer Stage (The "App" Look)
*   **Forced Centering:** `#viewer-container` uses `display: flex; align-items: center; justify-content: center;` with `padding: 40px;`.
*   **Clean Workspace:** Scrollbars are explicitly hidden on the container (`scrollbar-width: none;`).
*   **Stage Definition:** `#viewer-stage` is fixed at `1024px x 640px` with a `box-shadow: 0 0 50px rgba(0, 0, 0, 0.4)` and `border-radius: 8px`. This gives it a "contained" professional feel rather than a full-bleed window.
*   **Panorama Transitions:** `.panorama-layer` uses a `0.3s ease-in-out` transition for opacity and visibility, ensuring smooth "pan-then-swap" logic.

## 3. Linking Mode (Interactive Feedback)
*   **Yellow Crosshair Cursor:** A custom SVG cursor (`stroke='%23FFFF00'`) is forced on the canvas and drag-fix area during linking.
*   **Marching Ants Animation:** `.line-marching-ants` uses `stroke-dasharray: 10, 5` and a keyframe animation (`marching-ants`) to show active path creation.
*   **Cursor Dot Blink:** `.cursor-dot-blinking` uses a `0.8s` animation alternating between Gold and White with glow effects.
*   **Yellow Floor Rod:** The yellow dashed line connecting the last confirmed point to the mouse.

## 4. Hotspot UI (The "Arc" System)
*   **Multi-Layer SVG Arrows:** Hotspots use custom SVGs with linear gradients (`#hsG_idx`) and double paths for "glow-top" and "glow-bottom" effects.
*   **Interactive Controls:**
    *   `.hotspot-delete-btn`: A red circular button (`#dc3545`) with a `0.3s` pop-in animation using `cubic-bezier(0.175, 0.885, 0.32, 1.275)`.
    *   `.hotspot-controls`: A flex container at the bottom of the arrow holding `Forward` and `Return` toggles.
    *   **Color-Coded Toggles:** Active Forward is Teal (`#059669`), Active Return is Orange (`#ea580c`).
*   **Line Pulse:** `.line-pulse` animation on the path lines (`2s ease-in-out`) that modulates `stroke-width` and `stroke-opacity`.

## 5. Premium Modals (`.modal-box-premium`)
*   **Gradients:** `linear-gradient(to bottom, #001a38 0%, #002a70 50%, #003da5 100%)`.
*   **Animations:** Scale transition from `0.95` to `1.0` using `cubic-bezier(0.34, 1.56, 0.64, 1)`.
*   **Buttons:** `.modal-btn-premium` uses a background of `rgba(255, 255, 255, 0.08)` with a `transform: translateY(-2px)` hover effect and an inner inset shadow.

## 6. Simulation Mode (Lockdown Behavior)
*   **Grayscale Dimming:** During auto-pilot, the sidebar is dimmed using `filter: grayscale(100%); opacity: 0.85;`.
*   **Pulsing Stop Button:** The simulation toggle pulses red (`#dc3545`) with a specific `@keyframes pulse-stop` that expands a red shadow ring.
*   **Selective Visibility:** Informational buttons (category, label) stay `opacity: 1`, while editing buttons (add-link) are dimmed to `0.4`.

## 7. Floor Navigation
*   **The Circle UI:** `.floor-circle` uses a `Cobalt Blue (#0047AB)` background when active with a `1.15` scale and a `15px` cobalt glow.
*   **Hover State:** Unselected floors turn Navy with an `Orange (#f97316)` border and a slight lift.

## 8. Toast Notifications
*   **Sliding Entrance:** Toasts use `transform: translateX(30px)` and slide to `0` with a `0.5s` cubic-bezier entrance.
*   **Type-Specific Borders:** Error (Red), Success (Emerald), Warning (Amber) with translucent dark backgrounds.
