# 🛠️ Project Development Requirements

This project utilizes a high-performance ReScript (Frontend) and Rust (Backend) stack. The following tools are required for a full development environment.

## 📦 Core Dependencies
- **Node.js**: v18.0.0 or higher (https://nodejs.org/)
- **npm**: v9.0.0 or higher
- **Rust**: Stable toolchain (https://rustup.rs/)
- **Cargo**: Included with Rust

## ⚙️ System Utilities
- **ffmpeg**: Required for image and video processing pipeline.
- **fswatch**: Required for `project-guard.sh` to monitor codebase health in real-time.
- **ripgrep (rg)**: Recommended for high-speed codebase searching.
- **fd (fd-find)**: Recommended for fast file finding.

## 🌐 Web Application Stack
- **ReScript Compiler**: Installed via npm (`rescript`)
- **Rsbuild**: Build tool for the frontend
- **Concurrently**: Manages parallel dev processes (RES, BACK, FRONT)
- **Cargo Watch**: Recommended for backend hot-reloading (`cargo install cargo-watch`)
