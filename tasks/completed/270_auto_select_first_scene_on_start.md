# Task 270: Auto-select First Scene on Start Building

## Objective
When the user clicks "Start Building" after uploading images, the first scene should be automatically selected and displayed in the viewer.

## Context
Currently, the viewer might remain empty or not properly initialized with the first scene after the upload modal closes and the user enters the building mode.

## Steps
1. Identify the "Start Building" action in the codebase.
2. Check why the first scene is not selected.
3. Implement auto-selection of the first scene if no scene is currently selected.
4. Verify the fix.
