# Completed Task Reports

This directory contains reports of completed tasks.

## Process
When a task is completed:
1. The task file is moved from `tasks/pending/` (or `tasks/active/`) to `tasks/completed/`.
2. The file is renamed to include the `_REPORT` suffix (e.g., `15_Backend_SingleZIP_Load_REPORT.md`).
3. The content is transformed to a report format:
   - Headers like `Task` become `Report`.
   - Sections like `Objective` become `Objective (Completed)`.
   - The report summarizes what was accomplished.

This ensures that only critical records of completed work remain in this folder.
